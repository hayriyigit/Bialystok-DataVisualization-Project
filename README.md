## Description
Project done for Data Visualization course at Bialystok University of Technology. Summer semester 2020

## Requirements
* NodeJs
* npm or Yarn
## Instructions 
* Clone the project
* Go to clone folder
* Run npm install / yarn install
* Run npm start / yarn start
* Go to http://localhost:3000 in your web browser
